Building your first repository

1. Create a new account with Github.

2. Download Github Desktop to your computer and install.

3. Click on file then New repository and save.

4. Go to VS Code and link it to Github Desktop.

5. Create a new folder which will store your Github repository documents.

6. The name of the new folder should be the same as the repository name.

7. On VS code, open the new folder created.

8. Begin writing your code on VS Code.

9. Save your code on VS code.

10. Go to Github desktop under the changes tab.

11. Write a commit message.
